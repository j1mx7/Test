### Toast Notification System
**Class**: `ToastNotification` + `NotificationManager`

**API**:
```python

# Info (blue icon)
notifications.notify_info(text, duration_ms=2500)

# Error (red X, manual dismiss)
notifications.notify_error(text, details=None)
```

**Features**:
- ‚úÖ Top-right positioning
- ‚úÖ Slide-down entrance animation
- ‚úÖ Slide-up exit animation
- ‚úÖ Auto-dismiss with timings
- ‚úÖ De-duplication (1s window)
- ‚úÖ Expandable error details
- ‚úÖ Keyboard accessible
- ‚úÖ Queue support (one at a time)


**Features**:
- ‚úÖ Frameless dark themed

- ‚úÖ 200ms fade-in animation
- ‚úÖ ESC/Enter keyboard support
