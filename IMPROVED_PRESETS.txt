HOW TO USE:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Load nvram.txt file
2. Go to Presets tab
3. Right panel: Select CPU Family (Intel/AMD)
4. Right panel: Navigate Basic ↔ Advanced (< > arrows)
5. Right panel: Enable preset toggles
6. Left table: Preview settings from enabled presets
7. Bottom: Click "Apply Config" to write changes
